---

## Key Insights
- Sales increased over time, **month by month**
- A small number of products generate most of the revenue
- Certain days of the week consistently perform better than others

---

## Project Workflow
1. Cleaned and preprocessed raw sales data using Python (pandas)
2. Created time-based features (month and day of the week)
3. Analyzed revenue trends and product performance
4. Built an interactive Power BI dashboard to visualize insights
